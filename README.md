## Dependencies

- React
- React-dom
- React-scripts
- Web-vitals
- Axios
- Node.js
- Express
- Cors
- Dotenv
- Helmet
- PG
- Socket.io
- WS

## Setup

Install dependencies with `npm install` in each respective `/frontend` and `/backend`.

## [Frontend] Running Webpack Development Server

```sh
cd frontend
npm start
```

## [Backend] Running Backend Servier

Read `backend/readme` for further setup details.

```sh
cd backend
npm start
```
